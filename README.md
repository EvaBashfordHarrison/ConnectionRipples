# Digital Echo

## Description
This project looks at connections in an online space. With no other information about whether you're alone or not, users are invited to click and drag across the screen to create a digital echo. If there is someone on the other side, they can repeat the action as a call and response. Different users are given different colours to distinguish identities. 

## Installation
npm install express, node app.js 

## Technology and hardware 
Built in p5.js which can run in most browsers - Chrome is preferred. Users will need either a trackpad from a laptop or an external mouse to play. To enjoy the connectivity, an internet connection is required. 

## Usage
A simple web app to be used on desktop. Mobile is possible with some issues on other screens.
